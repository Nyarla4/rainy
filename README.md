- 한자 부분이 내려오고, 발음과 뜻을 기입하는 형태의 산성비입니다
- 난이도는 N1~N5 중 1택합니다
- 속도는 빠름/보통/느림(임의라 보통도 빠르게 느낄 수 있음)의 셋입니다
- 뜻의 경우 아래 규칙을 따릅니다

| 항목 | 처리 규칙 |
|---|---|
| 괄호(()) | (직무로서) 취급하다" 의 경우 "직무로서 취급하다"와 "취급하다"가 답으로 처리 |
| 대괄호([]) | "에서[으로] 자랐음[자란 사람]" 의 경우 "에서 자랐음", "에서 자란 사람", "으로 자랐음", "으로 자란 사람" 모두가 답으로 처리 |
| 쌍점(:) | "마음속 깊이 느끼는 모양:정말" 의 경우 "정말"만 답으로 처리 |

- 발음 및 뜻은 네이버 일본어 사전 기준으로 기입하되, 일부 수정이 있을 수 있습니다
- **public으로 되어있으므로 발음이나 뜻에 대해서 개선사항 혹은 불만사항이 있는 경우 fork해서 push 바랍니다**
